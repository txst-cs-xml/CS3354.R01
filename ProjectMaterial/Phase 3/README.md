## How to Run the Project
Follow these instructions to compile and run the project:

1. **Clone the Repository**:  
   ```bash
   git clone [repository link]
   cd [repository folder]
   ```

2. **Compile the Code**:  
   ```bash
   javac -d bin src/*.java
   ```

3. **Run the Game**:  
   ```bash
   java -cp bin ChessGame
   ```
